# PM Templates & Work Samples
A collection of anonymized work artifacts from real product management projects.
**What's here:**

- **PRD_Market_Expansion_Template.md** — Reusable PRD template for market expansion (T&S tooling). Built during Allegro's CZ/SK expansion. Includes OKRs, checklist, decision log, Jira integration.

- **Analiza_Projektu_SOW_Kompletna.md** — Project status assessment + SOW estimation for a mobile app project. Includes risk analysis, resource gap identification, and corrected dependency sequencing.

- **Employee_Data_Anonymized.xlsx** — Transformation of unstructured employee data into scalable multi-view spreadsheet (HR, Accounting, Management Dashboard). Raw data included for before/after comparison.

- **AI_Narrative_Designer_Anonymized.docx** — AI persona design for customer service training. Includes interaction mechanics, escalation paths, and generated dialogs with prompt documentation.
